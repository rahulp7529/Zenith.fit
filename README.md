# GymFlow - Premium Workout Tracking App

A luxury fitness application built with React, Vite, Tailwind CSS, and Three.js. Features advanced UI interactions, 3D muscle visualization, and intelligent workout tracking.

## ✨ Features

- **Weekly Split Navigation** - Smooth horizontal day selector with active state glow
- **Exercise Tracking** - Premium glass design cards with progress indicators
- **Set Logging** - Checkbox-based set completion with smooth animations
- **3D Muscle Visualization** - Interactive Three.js body with muscle highlighting
- **Clickable Body Parts** - Select muscle groups to discover exercises
- **Micro-interactions** - Smooth Framer Motion animations throughout
- **Premium Dark Theme** - CRED-inspired glassmorphism design
- **Local State Management** - React hooks for persistent workout data

## 🎨 Design System

- **Dark Theme**: True blacks (#000, #0A0A0A, #111)
- **Accent Color**: Electric Blue (#0066FF)
- **Effects**: Glassmorphism, subtle shadows, soft glow
- **Typography**: Bold, modern sans-serif with high contrast
- **Spacing**: Generous padding and breathing room

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Advanced animations
- **Three.js** - 3D visualization
- **JavaScript ES modules** - Modern JavaScript
